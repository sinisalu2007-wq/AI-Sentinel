# AI-Sentinel 🛡️
### AI-Powered Network Anomaly Detection for African Infrastructure
![AI-Sentinel Dashboard Detection](docs/images/detection_screenshot.png)
**AI-Sentinel** is a cybersecurity solution designed to detect sophisticated network threats using Machine Learning. 

## 🚀 Features
- **Real-time Traffic Sniffing:** Built with Python and Scapy.
- **Behavioral Analysis:** Uses Isolation Forest algorithms to find "outliers" (hacker activity).
- **Web Dashboard:** A PHP-based interface for local IT administrators.

## 🛠️ Built With
- **Kali Linux** (Development Environment)
- **Python 3.8+** (AI Engine)
- **Scikit-Learn** (Machine Learning)
- **PHP/MySQL** (Dashboard)
